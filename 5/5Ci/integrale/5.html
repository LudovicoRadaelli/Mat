Regole di integrazione (generalizzate)

  • \[ \int f(x)^{\alpha} f'(x) \,\,dx = \dfrac{f(x)^{\alpha + 1}}{\alpha + 1} + c \quad \text{con} \quad \alpha \neq -1 \]

  • \[ \int \dfrac{f'(x)}{f(x)} \,\, dx = ln\left(|f(x)|\right) + c \]

  • \[ \int e^{f(x)}f'(x) \,\, dx = e^{f(x)} + c \]

  • \[ \int sin(f(x))f'(x) \,\, dx = -cos(f(x)) + c \]

  • \[ \int cos(f(x))f'(x) \,\, dx = sin(f(x)) + c \]

  • \[ \int \dfrac{1}{cos^2(f(x))}f'(x) \,\, dx = tan(f(x)) + c \]

  • \[ \int \dfrac{1}{sin^2(f(x))}f'(x) \,\, dx = -cotan(f(x)) + c \]

  • \[ \int \dfrac{f'(x)}{\sqrt{1 -f(x)^2}} \,\, dx = arcsin(f(x)) + c \]

  • \[ \int \dfrac{f'(x)}{1 + f(x)^2} \,\, dx = arctan(f(x)) \]




Esercizio 1


Svolgere i seguenti integrali


  • \[ \int \left(4x^3 -6x\right)\left(x^4 -3x^2 +1\right)^3\,\,dx \]

    • Soluzione: \[ \dfrac{\left(x^4 -3x^2 +1\right)^4}{4} + c \]




  • \[ \int \dfrac{16x +1}{8x^2 +x -1}\,\,dx \]

    • Soluzione: \[ ln \left|8x^2 +x -1\right| + c \]




  • \[ \int -sin\left(x\right)e^{cos\left(x\right)} \,\,dx \]

    • Soluzione: \[ e^{cos\left(x\right)} + c \]




  • \[ \int \left(2x\,sin\left(x^2\right) - \dfrac{cos\left(ln\left(x\right)\right)}{x}\right) \,\,dx \]

    • Soluzione: \[ -cos\left(x^2\right) - sin\left(ln\left(x\right)\right) + c \]




  • \[ \int x\,\left(3x^2 + 2\right)^2 \,\,dx \]

    • Soluzione: \[ \dfrac{\left(3x^2 + 2\right)^3}{18} + c \]
