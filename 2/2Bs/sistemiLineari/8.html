FILA A


Esercizio 1


Svolgere il seguente sistema usando un metodo a scelta \[ \begin{cases} \left(3 - 2y\right)^2 - 4y\left(y - 1\right) + 5x = 1 - x \\\\ -2\left(1 + x\right) - 3\left(2x + 2y\right) = 2 \end{cases} \]


Soluzione: \(\left(-\dfrac{4}{5}\,; \dfrac{2}{5}\right)\)





Esercizio 2


Svolgere il seguente sistema usando il metodo di sostituzione \[ \begin{cases} \dfrac{2x -1}{5} - \dfrac{y + 1}{2} = \dfrac{x - y}{2} -\dfrac{x +3}{10} \\\\ \dfrac{1}{2} \,\left(x +1\right) - 3y = 2 \end{cases} \]


Soluzione: impossibile





Esercizio 3


Svolgere il seguente sistema usando il metodo di riduzione \[ \begin{cases} \dfrac{1}{3}\left(x - \dfrac{y +1}{2}\right) - \dfrac{1}{2}\left(\dfrac{x +1}{3} - y\right) = -\dfrac{5}{4} \\\\ \dfrac{1}{2}\left[x - 2\left(\dfrac{y -2}{3} + 1\right)\right] = 2x - \dfrac{1}{12} \end{cases} \]


Soluzione: \(\left(\dfrac{1}{2}\,; -3\right)\)





Esercizio 4


Svolgere il seguente sistema usando il metodo di riduzione \[ \begin{cases} 2x + y -z = 6 \\\\ x + y + z = 1 \\\\ x -y +2z = -5 \end{cases} \]


Soluzione: \(\left(1\,; \,\,2\,; \,\,-2\right)\)





Esercizio 5


Risolvere il seguente sistema letterale. \[ \begin{cases} 6x + ky = -1 + k \\\\ 5x -2ky = 4 \end{cases} \]

Stabilire per quali valori di \(k\) il sistema risulta

  • impossibile
  • indeterminato
  • determinato. In questo caso mostrare le soluzioni in funzione di \(k\).

Soluzione:

se \(k \neq 0\) allora \(\left(\dfrac{2k +2}{17}\,; \dfrac{5k -29}{17k}\right)\)

se \(k = 0\) allora l'equazione è impossibile





FILA B


Esercizio 1

Svolgere il seguente sistema usando un metodo a scelta \[ \begin{cases} \left(2x + 1\right)^2 -5\left(1 -y\right) = 4x^2 -y \\\\ 5x + \left(2x + 1\right)\left(y - 2\right) = 2xy + 1 \end{cases} \]


Soluzione \(\left(7\,; -4\right)\)





Esercizio 2

Svolgere il seguente sistema usando il metodo di sostituzione \[ \begin{cases} \dfrac{3x + 1}{2} - \dfrac{2y -1}{3} = \dfrac{2}{3} \left(2x -y\right) + \dfrac{2 + x}{6} \\\\ \dfrac{x}{3} - \dfrac{y}{2} = 1 \end{cases} \]


Soluzione: impossibile





Esercizio 3

Svolgere il seguente sistema usando il metodo di riduzione \[ \begin{cases} \dfrac{1}{3} \left(\dfrac{x-2}{4} - \dfrac{y + 1}{2}\right) = -\left(\dfrac{3x -2}{3} - \dfrac{y -3}{2}\right) + 3 \\\\ \dfrac{4x - \left(2y - x\right)}{2} = \dfrac{1 - y}{10} - \left(\dfrac{1}{5}x -\dfrac{23}{4}\right) \end{cases} \]


Soluzione: \(\left(2\,; -\dfrac{1}{2}\right)\)





Esercizio 4

Svolgere il seguente sistema usando il metodo di riduzione \[ \begin{cases} 2x + y + z = 1 \\\\ 4x -y +z = -5 \\\\ -x +y +2z = 5 \end{cases} \]


Soluzione: \(\left(-1\,; \,\,2\,; \,\,1\right)\)





Esercizio 5

Risolvere il seguente sistema letterale. \[ \begin{cases} kx + 4y = 1 - k \\\\ 5kx + 2y = 1 \end{cases} \]

Stabilire per quali valori di \(k\) il sistema risulta

  • impossibile
  • indeterminato
  • determinato. In questo caso mostrare le soluzioni in funzione di \(k\).


Soluzione:

se \(k \neq 0\) allora \(\left(\dfrac{k +1}{9k}\,; \dfrac{4 -5k}{18}\right)\)

se \(k = 0\) allora l'equazione è impossibile
