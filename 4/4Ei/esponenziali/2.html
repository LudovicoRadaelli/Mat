Esercizio 1


Risolvere le seguenti equazioni del tipo passaggio agli esponenti


  • \(49^{x +2} = \sqrt[3]{7^{x + 7}}\)


    Soluzione:\(\quad x = -1\)


  • \(8 \sqrt[4]{2^3} = 16^{2x}\)


    Soluzione:\(\quad x = \dfrac{15}{32}\)





Esercizio 2


Risolvere le seguenti equazioni del tipo per sostituzione


  • \(5^{2x} - 5^x = 5^{x-2} -\dfrac{1}{25}\)


    Soluzione:\(\quad x = 0 \quad\text{oppure}\quad x = -2\)


  • \(\dfrac{2}{3^x -1} = \dfrac{1}{3^x -5}\)


    Soluzione:\(\quad x = 2\)


  • \(2^{x+1} + 2^{3-x} = 17\)


    Soluzione:\(\quad x = -1 \quad\text{oppure}\quad x = 3\)
