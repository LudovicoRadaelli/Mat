Regole di integrazione


  • \[ \int x^{\alpha} \,\,dx = \dfrac{x^{\alpha + 1}}{\alpha + 1} + c \quad \text{con} \quad \alpha \neq -1 \]

  • \[ \int \dfrac{1}{x} \,\, dx = ln\left(x\right) + c \quad \text{con} \quad x \gt 0 \]

  • \[ \int e^{x} \,\, dx = e^{x} + c \]

  • \[ \int sin(x) \,\, dx = -cos(x) + c \]

  • \[ \int cos(x) \,\, dx = sin(x) + c \]

  • \[ \int \dfrac{1}{cos^2(x)} \,\, dx = tan(x) + c \]

  • \[ \int \dfrac{1}{sin^2(x)} \,\, dx = -cotan(x) + c \]

  • \[ \int \dfrac{1}{\sqrt{1 -x^2}} \,\, dx = arcsin(x) + c \]

  • \[ \int \dfrac{1}{1 + x^2} \,\, dx = arctan(x) + c \]


Regole di integrazione (generalizzate)

  • \[ \int f(x)^{\alpha} f'(x) \,\,dx = \dfrac{f(x)^{\alpha + 1}}{\alpha + 1} + c \quad \text{con} \quad \alpha \neq -1 \]

  • \[ \int \dfrac{f'(x)}{f(x)} \,\, dx = ln\left(|f(x)|\right) + c \]

  • \[ \int e^{f(x)}f'(x) \,\, dx = e^{f(x)} + c \]

  • \[ \int sin(f(x))f'(x) \,\, dx = -cos(f(x)) + c \]

  • \[ \int cos(f(x))f'(x) \,\, dx = sin(f(x)) + c \]

  • \[ \int \dfrac{1}{cos^2(f(x))}f'(x) \,\, dx = tan(f(x)) + c \]

  • \[ \int \dfrac{1}{sin^2(f(x))}f'(x) \,\, dx = -cotan(f(x)) + c \]

  • \[ \int \dfrac{f'(x)}{\sqrt{1 -f(x)^2}} \,\, dx = arcsin(f(x)) + c \]

  • \[ \int \dfrac{f'(x)}{1 + f(x)^2} \,\, dx = arctan(f(x)) \]

Svolgere i seguenti integrali


  • \[ \int \left(3x^5 - \dfrac{5}{x^6} + 5\,\sqrt[5]{x^3}\right)\,\,dx \]

    • Soluzione: \[ \dfrac{x^6}{2} + \dfrac{1}{x^5} + \dfrac{25}{8} \sqrt[5]{x^8} + c \]


  • \[ \int \left(5\,sin(x) - 6\,cos(x) - \dfrac{\sqrt{5}}{2}\,e^x\right)\,\,dx \]

    • Soluzione: \[ -5\,cos(x) -6\,sin(x) - \dfrac{\sqrt{5}}{2} + c \]


  • \[ \int \dfrac{3x -5x^2 + \sqrt[5]{x^{11}}}{x^2}\,\,dx \]

    • Soluzione: \[ 3\,ln(x) -5x + \dfrac{5}{6}\,\sqrt[5]{x^6} + c \]


  • \[ \int \left(\dfrac{7}{1 + x^2} - \dfrac{3}{2\sqrt{1 - x^2}}\right)\,\,dx \]

    • Soluzione: \[ 7\,arctan(x) -\dfrac{3}{2}\,arcsin(x) + c \]


  • \[ \int \left[\left(2x + 1\right)\,\left(x^2 + x\right) + x\right]\,\,dx \]

    • Soluzione: \[ \dfrac{x^4}{2} + x^3 + x + c \]


  • \[ \int \dfrac{2\,cos(x)}{1 + cos^2(x)} + \dfrac{sin(x)}{cos(x)} \,\,dx \]

    • Soluzione: \[ arctan(cos(x)) - ln\left|cos(x)\right| + c \]


  • \[ \int \left(4x^3 -3\,cos(x)\right)\left(-3\,sin(x) + x^4\right)^4\,\,dx \]

    • Soluzione: \[ \dfrac{\left(-3\,sin(x) + x^4\right)^5}{5} + c \]
