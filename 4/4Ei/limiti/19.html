Esercizio 1


La funzione \[ f(x) = \begin{cases} \sqrt{x + 7} \quad \text{se }\,\, x \leq 2 \\\\ \dfrac{1}{x -2}\quad \text{se }\,\, x > 2 \end{cases} \] è continua in \(x = 2\)?





Esercizio 2


La funzione \[ f(x) = \begin{cases} 2x -1 \quad \text{se }\,\, x \lt 5 \\\\ x^2 - 16 \quad \text{se }\,\, x \geq 5 \end{cases} \] è continua in \(x = 5\)?





Esercizio 3


La funzione \[ f(x) = \begin{cases} \dfrac{-6x^2 +7x -2}{1 -4x^2} \quad \text{se }\,\, x \leq \dfrac{1}{2} \\\\ \dfrac{-4x^2 -8x +5}{-2x +1}\quad \text{se }\,\, x > \dfrac{1}{2} \end{cases} \] è continua in \(x = \dfrac{1}{2}\)?





Esercizio 4


Determinare per quali valore del parametro \(a\) a funzione \[ f(x) = \begin{cases} 3x + 2b \quad \text{se } x \leq 5 \\\\ bx -1 \quad \text{se} x > 5 \end{cases} \] è continua in \(x = 5\)?





Esercizio 5


Determinare per quali valore del parametro \(a\) a funzione \[ f(x) = \begin{cases} \sqrt{x^2 +2x + a} \quad \text{se } x \leq -1 \\\\ 5x + 8 \quad \text{se} x > -1 \end{cases} \] è continua in \(x = -1\)?
